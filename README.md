# C Programming Lab - CSE107

This repository contains C programming lab exercises completed as part of the CSE107 course.

## Contents

- Basic C programs
- Control structures (if, loops)
- Functions
- Arrays and strings
- Problem-solving programs

## Folder Structure

Programs are organized as individual C files based on lab exercises.

## How to Run

1. Compile:
   gcc filename.c

2. Run:
   ./a.out

## Purpose

This repository demonstrates foundational programming skills and problem-solving using C.
